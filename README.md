 More Details (https://github.com/mauricius/vue-draggable-resizable)

<h3>Rotate Support</h3>

Adding Rotate Handler
#### handles
You can add 'rot' key for rotating.
* `rot` - Rotate
```html
<vue-draggable-resizable :handles="['tm','bm','ml','mr','rot']">
```
